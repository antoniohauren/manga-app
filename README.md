## Description
- The backend is the project itself
- The frontend is on the frontend folder


## Technologies
- backend
  - nestjs
  - prisma
  - postgreeSQL
- frontend
  - react
  - nextjs
- project
  - docker
  - docker-compose

## To run
copy and make the changes you want:
```
'.env.example' to '.env'
```

run:
```
docker compose up
```

open:

```
http://localhost:3000
```
